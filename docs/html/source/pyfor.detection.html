pyfor.detection module¶

class pyfor.detection.LayerStacking(cloud, n_jobs=1, chm_resolution=1, buffer_distance=0.5, first_pass_min_dist=3, first_pass_threshold_abs=3, veg_layers=(0, 1, 2), percentiles=(70, 80, 90, 100), weights=(2, 3, 4, 4), overlap_kernel_size=3, scnd_pass_min_dist=3, scnd_pass_threshold_abs=3)¶

Bases: object

An implementation of Ayrey et al. (2017) layer stacking algorithm.

__init__(cloud, n_jobs=1, chm_resolution=1, buffer_distance=0.5, first_pass_min_dist=3, first_pass_threshold_abs=3, veg_layers=(0, 1, 2), percentiles=(70, 80, 90, 100), weights=(2, 3, 4, 4), overlap_kernel_size=3, scnd_pass_min_dist=3, scnd_pass_threshold_abs=3)¶
Parameters:
  • cloud – The cloud object to detect on.
  • n_jobs – The number of threads to conduct KMeans.
  • chm_resolution – The CHM resolution for the initial detection.
  • buffer_distance – Distance to buffer points.
  • first_pass_min_dist – The minimum distance for initial top detection.
  • first_pass_threshold_abs – The absolute threshold for initial top detection.
  • veg_layers – Layers to be considered vegetation
  • percentiles – Percentiles to weight.
  • weights – Weights given to layers within provided percentiles.
  • overlap_kernel_size – The smoothing kernel size for the overlap map (in pixels).
  • scnd_pass_min_dist – The minimum distance for the final top detection.
  • scnd_pass_threshold_abs – The absolute threshold for initial top detection.
