classmethod from_dir(las_dir, n_jobs=1, get_bounding_boxes=True)¶

Wrapped function for producing a CloudDataFrame from a directory of las files. :param las_dir: :param n_jobs: The number of threads used to construct information about the CloudDataFrame. :param get_bounding_boxes: If True, builds the bounding boxes for each las tile by manually reading in the file and computing the bounding box. For very large collections this may be computationally costly, and can be set to False. :return:

par_apply(func, column='las_path', buffer_distance=0, *args)¶

Apply a function to each las path. Allows for parallelization using the n_jobs argument. This is achieved via joblib Parallel and delayed.

Parameters:
  • func – The user defined function, must accept a single argument, the path of the las file.
  • n_jobs – The nlumber of threads to spawn, default of 1.
  • column – The column to apply on, will be the first argument to func
  • buffer_distance – The distance to buffer and aggregate each tile.

...                    'year': [2012, 2014, 2013, 2014],

1  4      40    2014

Set the index to become the ‘month’ column:

>>> df.set_index('month')
       sale  year
month
1      55    2012
4      40    2014
7      84    2013
10     31    2014

Create a multi-index using columns ‘year’ and ‘month’:

>>> df.set_index(['year', 'month'])
            sale
year  month
2012  1     55
2014  4     40
2013  7     84
2014  10    31

Create a multi-index using a set of values and a column:

>>> df.set_index([[1, 2, 3, 4], 'year'])
